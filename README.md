# GitHubリポジトリ検索アプリ

GitHubのリポジトリを検索できます。<br>
検索されたリポジトリをタップすると、そのリポジトリの詳細を見ることができます。<br>
GitHubのアカウントでログインしたユーザーのアイコンをタップすると、ユーザーの詳細やリポジトリ一覧を見ることができます。<br>
